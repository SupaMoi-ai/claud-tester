# BRO CODE — Roadmap

A scratchpad of things we could build next. Pick whatever feels most useful — none of this is committed, nothing here blocks anything else. Group by theme.

## Content depth

- [ ] More **situation chips** (e.g. *She's anxious about work*, *She's not sleeping*, *We had a long-distance week*).
- [ ] **Multi-cycle history view** — a horizontal scroll of the last 6 cycles with logged events overlaid, so you can spot patterns ("our worst fights always cluster in late luteal").
- [ ] **Conflict-pattern tracker** — when you log ⚡ CONFLICT, prompt 3 quick taps (who started it / phase / topic). Over time, show patterns ("60% of your conflicts are in late luteal, around money").
- [ ] **Phase-specific FAQ deeper dives** (sleep changes, libido changes, hunger changes, exercise tolerance) — written like a coach, not a textbook.
- [ ] **Mini-articles** triggered by what you log — log 3 conflicts in a row → "// here's why this might be happening" reading.

## Her side

- [ ] **Her mood input** — give her a private one-tap "today felt: 😊 😐 😢 😡" surface. Stored locally. Affects today's tip without exposing what she tapped.
- [ ] **Shared profile** (optional) — let her edit her own profile fields. Defuses the "are you tracking me?" concern.
- [ ] **Female-perspective coach lines** — a few tips voiced like a sister/friend who's been there, not a "bro." Toggle in profile.
- [ ] **Period-confirm button** — when an actual period starts, one tap moves the start date so the math stays accurate.

## Calmer bro

- [ ] **Breathing animation** — a 4-2-6 box-breathe visual circle that pulses for 30 seconds inside the "I'm spiraling" panel.
- [ ] **Phase-specific lo-fi / Spotify playlists** — tap a phase, get a 30-min playlist link tuned to the mood.
- [ ] **Weekly wins recap** — every Sunday, surface "you showed up X times this week. Specifically: ... " using the log entries.
- [ ] **"You're tired too" nudges** — every ~2 weeks, surface a one-liner reminding the bro to take care of himself.

## Plumbing

- [ ] **Cloud backup with E2E encryption** — opt-in. Profile + log syncs across devices. Encrypted client-side so no one (incl. us) can read it.
- [ ] **PIN / Face ID lock** — protect the app contents from prying eyes.
- [ ] **Import from Clue / Flo** — let her existing cycle history populate the profile.
- [ ] **Apple Watch complication / glance** — show today's phase + one-line move at a glance.
- [ ] **iCal export** — export the next 6 months of phase markers + alerts to her or his calendar.

## Polish

- [ ] **Onboarding tour** — first run walks through the 5 tabs in 60 seconds.
- [ ] **Anniversary / important-date tracking** — birthday, anniversary, "the day we met" → custom alerts.
- [ ] **Custom alerts** — let the bro set his own ("text mom on the 1st", "buy flowers on date").
- [ ] **Dark-humor intensity slider** — 1 (serious) to 5 (locker room). Default 3.
- [ ] **Voice notes** — record yourself saying the apology line so you can practice the tone before delivering it for real.
- [ ] **Streak milestones** — 7, 30, 90, 365 days of logging → unlock badges + a one-line milestone message.
- [ ] **Phase-aware emoji + theme** — Calendar tabs subtly tint to the current phase color.

## Bro self-care

- [ ] **Refuel tracker** — quick log: sleep, exercise, alone time. Surfaces "you're running low on Y" gently.
- [ ] **"Take your guy time" reminder** — if no `me-time` log entry in 7 days, suggest it.
- [ ] **Real-talk Sunday** — once a week, a longer-form "where's your head at?" prompt for the bro himself.

## Stretch / experimental

- [ ] **AI-generated dynamic tips** — same bro voice but trained on your specific patterns + recent logs.
- [ ] **Anonymous community of bros** — share wins, get advice, lurk. Heavily moderated.
- [ ] **Real coach connect** — paid link to talk to an actual therapist / coach who specializes in relationships + cycles.
- [ ] **Couples mode** — both partners install, opt-in shared phase view, mutual mood pulse.
